# Running PHPUnit Tests

To run all tests in this project:

### Clone this repository

```git
git clone https://github.com/itsfirestorm/M07_UF1_POO_Bank.git
cd M07_UF1_POO_Bank
```

## Run the tests:

### Linux / macOS

```bash
./vendor/bin/phpunit ./tests/*
```

### Powershell / CMD

```powershell
.\vendor\bin\phpunit .\tests\*
```
